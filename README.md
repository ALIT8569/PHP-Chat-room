# 纯php聊天室

## 功能特点

- **实时预览**: 发送图片链接会自动生成缩略图，缩略图点击可放大查看
- **实时聊天**: 聊天实时性不错
- **响应式设计**: 完美适配各种设备
- **后台管理**:带有后台管理，支持禁言账号，支持查看账号访问ip，支持强制下线账号，支持搜索账号，支持设置公告
- **聊天记录**:支持搜索历史记录，点击搜索结果可跳至聊天记录所在位置
